# 🚇 天津地铁最优路径检索系统

基于Dijkstra算法实现的天津地铁最优路径查询系统，支持任意两个站点之间的最短路径计算。

## 🎯 功能特点
- ✅ 完整的天津地铁线路数据（40个站点）
- ✅ Dijkstra最短路径算法实现
- ✅ 前端交互界面（HTML/CSS/JavaScript）
- ✅ 响应式设计，支持移动端
- ✅ 模拟加载动画和进度条
- ✅ 详细的结果展示

## 🛠️ 技术栈
- **后端算法**：C语言（Dijkstra算法）
- **前端界面**：HTML5、CSS3、JavaScript
- **开发工具**：VS Code、GCC编译器
